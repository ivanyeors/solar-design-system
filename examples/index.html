Usage in Code

Import and use tokens in your application:

// Import all tokens
import tokens from './src/tokens';

// Or import specific token categories
import { optionColors, themes } from './src/tokens';

// Usage example
element.style.backgroundColor = themes['color.fill.brand-rest'];
element.style.padding = optionScale['base.padding.size-5'];
